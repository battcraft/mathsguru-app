# MathsGuru AI - Topic 2: Geometry (Lines, Points, Rays) ## 1. Point (Bindu) - **Concept:** An exact position. No length, no width. - **Hinglish:** "Point matlab sirf ek jagah — bas ek chhota sa dot." - **Examples:** Pencil dot, Google Maps pin, Star in the sky. - **Visual:** A glowing Gulabi Pink dot. ## 2. Line (Rekha) - **Concept:** Goes on forever in both directions. - **Symbol:** ↔ AB - **Hinglish:** "Line ka koi end nahi hota — dono side chalta hi rehta hai." - **Examples:** Endless straight road, Railway tracks. - **Visual:** Line with arrows on both ends. ## 3. Line Segment (Rekha Khand) - **Concept:** Part of a line with fixed start and end points. - **Symbol:** AB̅ - **Hinglish:** "Jiska start bhi hai aur end bhi — woh line segment hai." - **Examples:** Matchstick, Edge of a ruler. - **Visual:** Line with dots on both ends. ## 4. Ray (Kiran) - **Concept:** Starts at one point, goes forever in one direction. - **Symbol:** → AB - **Hinglish:** "Ray ka ek starting point hota hai, par aage kabhi khatam nahi hoti." - **Examples:** Sun rays, Torch beam, Arrow. - **Visual:** Line with one dot and one arrow. ## Super Simple Summary - **Point:** Sirf ek dot. - **Line:** Dono taraf endless. - **Line Segment:** Start + End dono. - **Ray:** Start hai, end nahi. ## Probing Questions (Socho) - Road seedhi hoti hai ya curved? Kyun? - Parallel lines real life mein kahan dikhti hain? (e.g., Railway tracks)
